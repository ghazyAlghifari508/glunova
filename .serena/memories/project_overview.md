# Project Overview: Glunova

Glunova is a comprehensive diabetes management platform connecting patients, doctors, and admins.

## Purpose
- Connect patients with doctors for consultations.
- Provide structured health education.
- AI-based nutrition analysis (Glunova Vision).
- Personalized health roadmaps.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4 + CSS Custom Properties
- **Backend**: Supabase (Auth, Database, Storage)
- **AI**: OpenAI/OpenRouter
- **Components**: Radix UI, Lucide React, Recharts
- **Languages**: TypeScript

## Target Users
- **Patients**: Access dashboards, education, AI vision, consultations, roadmaps.
- **Doctors**: Manage consultations, schedules, and earnings.
- **Admins**: Approve doctors, manage content, view analytics.
